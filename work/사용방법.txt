2. 주요 메뉴
================================================================================

■ 대시보드
  - 월별 요약: 지출, 수입, 순지출 한눈에 확인
  - 카테고리별 차트: 어디에 돈을 많이 쓰는지 시각화
  - 최근 6개월 추이: 지출/수입 트렌드 분석

■ 거래내역
  ▶ 거래 추가하기
    1. "거래 추가" 버튼 클릭
    2. 필수 정보 입력:
       - 날짜
       - 계정 (카드/현금/은행)
       - 금액
       - 유형 (지출/수입)
       - 카테고리
    3. 선택 정보:
       - 가맹점명
       - 메모
       - 태그 (예: #출장, #가족)

  ▶ 거래 수정/삭제
    - 거래 항목 클릭 → 수정 또는 삭제

  ▶ 필터링
    - 날짜 범위 선택
    - 카테고리별 필터
    - 계정별 필터
    - 검색 기능 (가맹점명, 메모)

■ 계정 관리
  ▶ 계정 추가 (카드, 현금, 은행)
    1. "계정 추가" 버튼
    2. 정보 입력:
       - 계정 이름 (예: "삼성카드", "현금", "신한은행")
       - 유형 (card/cash/bank)
       - 카드 마지막 4자리 (선택)
       - 색상 (UI 구분용)

■ 카테고리
  ▶ 기본 카테고리 (자동 생성됨)
    - 식비, 카페/간식, 교통, 주거/관리, 통신
    - 쇼핑, 여가, 의료, 교육, 수입, 기타

  ▶ 카테고리 추가/수정
    - 새 카테고리 만들기
    - 색상 지정으로 구분

3. 고급 기능
================================================================================

■ 파일 가져오기 (Import)
  ▶ 지원 형식: CSV, Excel (XLS, XLSX)

  ▶ 가져오기 과정
    1. 메뉴: "가져오기" 클릭
    2. 파일 선택 (카드 명세서 CSV/Excel)
    3. 컬럼 매핑:
       - 날짜 → 날짜 컬럼
       - 금액 → 금액 컬럼
       - 가맹점 → 상호명 컬럼
       - 카테고리 → 카테고리 컬럼 (선택)
    4. 미리보기 확인
    5. "가져오기" 실행

  ▶ 팁
    - 신용카드 명세서 다운로드 → CSV로 저장 → 가져오기
    - 중복 체크 기능으로 같은 거래 방지

■ 영수증 관리
  1. "영수증" 메뉴
  2. "영수증 추가" 클릭
  3. 사진 업로드 또는 촬영
  4. OCR 자동 인식 (금액, 날짜 추출)
  5. 거래내역과 연결

■ 자동 분류 규칙
  ▶ 규칙 만들기
    1. "규칙" 메뉴
    2. "규칙 추가"
    3. 설정:
       - 패턴: "스타벅스" 포함
       - 대상 필드: 가맹점명
       - 자동 카테고리: "카페/간식"
       - 우선순위 설정

  ▶ 예시
    패턴: "CU", "GS25", "세븐일레븐"
    → 카테고리: "쇼핑" 또는 "식비"

    패턴: "카카오T", "택시"
    → 카테고리: "교통"

■ 예산 관리
  1. "예산" 메뉴
  2. 월별 카테고리 예산 설정
     - 식비: 50만원
     - 교통: 10만원
     - 쇼핑: 30만원
  3. 진행률 자동 표시
  4. 예산 초과 시 경고

4. 데이터 관리
================================================================================

■ 내보내기 (Export)
  ▶ 일별 내보내기
    - 날짜 선택 → Excel 다운로드

  ▶ 월별 내보내기
    - 월 선택 → 전체 거래내역 Excel

■ 백업
  ▶ 자동 백업 위치
    Windows: C:\Users\사용자명\AppData\Roaming\가계부\dev.db

  ▶ 수동 백업
    1. 위 경로의 dev.db 파일 복사
    2. 안전한 곳에 저장 (USB, 클라우드)
    3. 정기적으로 백업 (주 1회 권장)

  ▶ 복원
    1. 백업한 dev.db 파일을 원래 위치에 복사
    2. 앱 재시작

5. 팁과 요령
================================================================================

■ 효율적인 사용법

  ▶ 1. 태그 활용
    #출장 - 출장 관련 지출
    #가족 - 가족 행사
    #선물 - 선물 구매
    #정기 - 정기 결제 (구독료 등)

  ▶ 2. 정기 결제 관리
    - 매월 반복되는 지출은 메모에 "정기" 표시
    - 예산 계획 시 정기 지출부터 고려

  ▶ 3. 일일 기록 습관
    - 하루 마지막에 5분 투자
    - 당일 지출 빠짐없이 기록
    - 영수증 사진 바로 업로드

  ▶ 4. 월말 리뷰
    - 대시보드에서 월간 지출 패턴 확인
    - 예산 대비 실제 지출 비교
    - 다음 달 예산 조정

■ 활용 시나리오

  ▶ 시나리오 1: 신용카드 명세서 정리
    1. 카드사에서 CSV 다운로드
    2. 가계부 앱에서 "가져오기"
    3. 자동 분류 규칙으로 카테고리 자동 지정
    4. 누락/오류 항목만 수동 수정

  ▶ 시나리오 2: 가족 공유 지출 관리
    1. Portable 실행 파일을 USB에 저장
    2. 각 가족 구성원 PC에 복사
    3. dev.db 파일을 클라우드 동기화 폴더에 저장
    4. 모두가 같은 데이터 공유

  ▶ 시나리오 3: 월말 결산
    1. 대시보드에서 월간 요약 확인
    2. 카테고리별 지출 차트 분석
    3. 예산 초과 항목 체크
    4. Excel 내보내기로 보고서 작성

6. 문제 해결
================================================================================

■ 자주 묻는 질문

  Q: 비밀번호를 잊어버렸어요
  A: 데이터베이스 파일(dev.db)을 삭제하고 재시작하면 초기화됩니다.
     (데이터 손실 주의!)

  Q: 거래가 중복으로 들어갔어요
  A: 거래내역에서 중복 항목 찾아서 삭제

  Q: 카테고리를 잘못 지정했어요
  A: 거래 클릭 → 수정 → 카테고리 변경

  Q: 가져오기가 안 돼요
  A: CSV 파일이 UTF-8 인코딩인지 확인
     Excel에서 "CSV UTF-8"로 저장

  Q: 데이터를 다른 PC로 옮기고 싶어요
  A: dev.db 파일을 복사하여 새 PC의 같은 위치에 붙여넣기

7. 단축키 및 팁
================================================================================

■ 추천 워크플로우
  1. 매일 저녁 5분: 당일 지출 입력
  2. 주말 10분: 주간 영수증 정리
  3. 월말 30분: 월간 리뷰 및 다음 달 예산 계획

■ 데이터 안전 수칙
  - 주 1회 백업 (금요일 저녁 권장)
  - 클라우드 자동 동기화 설정
  - 중요 시점(연말 등) 수동 백업

■ 성능 팁
  - 오래된 거래는 월별 Excel로 내보내기 후 삭제
  - 영수증 사진은 압축하여 저장
  - 정기적으로 불필요한 태그/메모 정리
